# taskmaster-buddy
A useful App for all the Little Alex Hornes out there

A dual-screen Electron desktop application built for hosting and producing live Taskmaster events. This app acts as a production control board, allowing the host to seamlessly cast videos, images, web embeds, and a live scoreboard to a secondary display (like a projector or TV) without exposing the UI or desktop to the audience.

Prerequisites
Download and install node.js 
https://nodejs.org/en/download

Within the folder where the GH source is downloaded to (and extracted)

npm install  
npm install electron --save-dev  
npm start  


That's it, for now.

npm run package

To package up the project into an executable  
npm install electron-packager --save-dev  

Currently Windows Supported only
